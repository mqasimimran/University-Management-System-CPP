# University Management System (UMS)
**Developed by:** Muhammad Qasim Imran
**University:** University of Management and Technology (UMT)

## 🚀 Project Overview
This project is a comprehensive simulation of a university ecosystem, built to demonstrate high-level software architecture using **C++**. [cite_start]It manages complex relationships between departments, faculty, students, and campus infrastructure [cite: 9-12].

## 🛠 Core OOP Features Implemented
[cite_start]This system serves as a practical application of the six pillars of Object-Oriented Programming[cite: 13]:
* [cite_start]**Encapsulation:** Data and behaviors are bundled within classes like `Student`, `Faculty`, and `Department` [cite: 14-15].
* [cite_start]**Inheritance:** The `Student`, `Faculty`, and `Staff` classes inherit shared attributes from the `Person` base class [cite: 16-18, 177].
* [cite_start]**Polymorphism:** Demonstrated through method overriding of the `display()` function across the class hierarchy [cite: 19-20].
* [cite_start]**Abstraction:** Hides internal data complexity by providing clear interfaces via the `display()` method [cite: 21-23].
* [cite_start]**Composition:** Modeled physical relationships, such as a `Classroom` containing a `Building` [cite: 24-25].
* [cite_start]**Aggregation:** Represented "has-a" relationships, such as a `University` containing multiple `Departments` [cite: 26-27].
