# University Management System (UMS)
**Developed by:** Muhammad Qasim Imran
**University:** University of Management and Technology (UMT)

## 🚀 Project Overview
This project is a comprehensive simulation of a university ecosystem, built to demonstrate high-level software architecture using **C++**. It manages complex relationships between departments, faculty, students, and campus infrastructure.

## 🛠 Core OOP Features Implemented
This system serves as a practical application of the six pillars of Object-Oriented Programming:
* **Encapsulation:** Data and behaviors are bundled within classes like `Student`, `Faculty`, and `Department`.
* **Inheritance:** The `Student`, `Faculty`, and `Staff` classes inherit shared attributes from the `Person` base class.
* **Polymorphism:** Demonstrated through method overriding of the `display()` function across the class hierarchy.
* **Abstraction:** Hides internal data complexity by providing clear interfaces via the `display()` method.
* **Composition:** Modeled physical relationships, such as a `Classroom` containing a `Building`.
* **Aggregation:** Represented "has-a" relationships, such as a `University` containing multiple `Departments`.

## 💻 Tech Stack
* **Language:** C++
* **Libraries:** Standard Template Library (STL) - Vectors
* **Design Pattern:** Interactive Menu-Driven System
